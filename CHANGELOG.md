# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/), and this project adheres
to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [2.0.1] - 2026-04-13

### Fixed

-   Translation for key mappings.

## [2.0.0] - 2026-04-13

### Added

-   Ported to Minecraft 26.1.2.

## [1.0.6] - 2025-09-12

### Fixed

-   Support for Refined Storage v2.0.0-beta.12.

## [1.0.5] - 2025-08-22

### Fixed

-   Fixed the Wireless Crafting Grid being unable to be charged on Fabric.

## [1.0.4] - 2025-08-05

### Fixed

-   Fixed crash when closing a Wireless Crafting Grid that became available after opening the GUI.

## [1.0.3] - 2025-08-03

### Fixed

-   Support for Refined Storage v2.0.0-beta.6.

## [1.0.2] - 2025-07-31

### Fixed

-   Support for Refined Storage v2.0.0-beta.5.

## [1.0.1] - 2025-07-23

### Fixed

-   Support for Refined Storage v2.0.0-beta.3.

## [1.0.0] - 2025-03-28

### Fixed

-   Support for Refined Storage v2.0.0-beta.1.

## [0.1.2] - 2025-01-28

### Fixed

-   Support for Refined Storage v2.0.0-milestone.4.12.

## [0.1.1] - 2024-12-08

## [0.1.0] - 2024-11-24

### Added

-   Wireless Crafting Grid

### Changed

-   You can now configure the energy usage of the Wireless Crafting Grid's recipe transfer and autocrafting.
-   You can now use the Wireless Crafting Grid without being connected to a network.
